LaTeX

Manutenção do modelo em LaTeX para trabalhos acadêmicos de acordo com as normas da UTFPR-PG.

Atividades

  • Documentação utilizando Literate Programming.
  • Ler as normas buscando por pontos que podem ser melhorados.
  • Implementação de novas funcionalidades utilizando LaTeX.

Habilidades necessárias

  • Programação básica (if, else, macros).
  • Noções de LaTeX.
  • Preocupação com detalhes e organização de código.

O que você poderá aprender com isso?

  • A técnica de Literate Programming.
  • Como escrever um pacote LaTeX.
  • Formatação de textos usando LaTeX.

Minicursos

Preparação e apresentação dos minicursos sobre Linux e LaTeX para os calouros do próximo semestre.

Atividades

  • Preparação de material teórico e prático sobre Linux e/ou LaTeX.
  • Apresentação dos minicursos na segunda/penúltima semana de aulas.

Habilidades necessárias

  • Capacidade de expressar-se.
  • Vontade de difundir conhecimento.
Linux
  • Comandos básicos.
  • Sistema de arquivos.
  • Principais distros.
  • Shell script básico.
LaTeX
  • Workflow básico.
  • Pacotes.
  • Ambientes.

O que você poderá aprender com isso?

  • Ajudar os novos colegas a aprender ferramentas úteis.
  • Adquirir experiência em ministrar aulas.

thinarch

Manutenção do servidor de thin-clients que é utilizado na parte prática dos minicursos.

Atividades

  • Melhorias no script de gerenciamento do servidor.
  • Criação/manutenção de um pacote de instalação para o servidor.

Habilidades necessárias

  • Linux.
  • Shell script.

O que você poderá aprender com isso?

  • Criação de pacotes para distribuições Linux.
  • Gerenciar um sistema de clientes burros(thin-clients).
  • Sistemas de arquivos NFS.
  • Configuração de DHCP.
  • Configuração de TFTP.
  • Shell script.
