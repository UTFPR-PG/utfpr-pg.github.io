UTFPR-PG On GitHub

O que o grupo UTFPR-PG tem aqui no Github?

Nós descrevemos aqui quais são os projetos desenvolvidos através do grupo da UTFPR campus Ponta Grossa. Estes são só alguns detalhes principais de cada projeto, como o que é feito, o que você teoricamente precisa saber para ajudar e quais as vantagens e desvantagens.

Para maiores informações use as formas de contato no fim da página ou seja destemido e nos pergunte pessoalmente!

Minicursos

Preparação e apresentação dos minicursos sobre Linux e LaTeX para os calouros do próximo semestre.

Atividades

  • Preparação de material teórico e prático sobre Linux e/ou LaTeX.
  • Apresentação dos minicursos na segunda/penúltima semana de aulas de cada semestre.

Habilidades necessárias

  • Capacidade de expressar-se.
  • Vontade de difundir conhecimento.
Linux
  • Comandos básicos.
  • Sistema de arquivos.
  • Principais distros.
  • Shell script básico.
LaTeX
  • Workflow básico.
  • Pacotes.
  • Ambientes.

O que você poderá aprender com isso?

  • Adquirir experiência em ministrar aulas, se expressar de maneira mais fácil e diminuir o nervosismo em apresentações são alguns exemplos.

Vantagens

  • Ajudar novos colegas a aprender ferramentas úteis.

Desvantagens

  • Sair do anonimato entre colegas de curso, professores e calouros.

LaTeX

Manutenção do modelo em LaTeX para trabalhos acadêmicos de acordo com as normas da UTFPR-PG.

Atividades

  • Documentação utilizando Literate Programming.
  • Ler as normas buscando por pontos que podem ser melhorados.
  • Implementação de novas funcionalidades utilizando LaTeX.

Habilidades necessárias

  • Programação básica (if, else, macros).
  • Noções de LaTeX.
  • Preocupação com detalhes e organização de código.

O que você poderá aprender com isso?

  • A técnica de Literate Programming.
  • Como escrever um pacote LaTeX.
  • Formatação de textos usando LaTeX.
  • As normas de trabalhos acadêmicos da UTFPR.
  • Se perguntar porquê existem essas normas!

thinarch

Manutenção do servidor de thin-clients que é utilizado na parte prática dos minicursos.

Atividades

  • Melhorias no script de gerenciamento do servidor.
  • Criação/manutenção de um pacote de instalação para o servidor.

Habilidades necessárias

  • Linux.
  • Shell script.

O que você poderá aprender com isso?

  • Criação de pacotes para distribuições Linux.
  • Gerenciar um sistema de clientes burros (thin-clients).
  • Sistemas de arquivos NFS.
  • Configuração de DHCP.
  • Configuração de TFTP.
  • Shell script.
